How to Use This Quiz

Playing Offline

Option A: Double-click index.html and open in your browser (file://) — works directly!

Option B: For phone access, run this command in terminal from the quiz folder:

python -m http.server 8000

Then use http://<laptop-ip>:8000 on phones connected to the same WiFi/hotspot. Generate a QR code from this URL for easy access.

Quiz Features

  • Test your knowledge across 4 wellness categories: Yoga, Sound Healing, Breathwork, and Meditation
  • Questions are randomly selected from a bank of 100 questions
  • Immediate feedback with explanations after each answer
  • Optional timer and user information collection
  • Printable certificate upon completion
  • Admin panel for managing questions and exporting results

Accessibility

This quiz supports keyboard navigation, screen readers, and has high contrast design for accessibility.
